# Vantagics（万策）- 智能数据分析平台

## 🎯 产品定位

**Vantagics（万策）** 是一款面向企业级用户的智能数据分析桌面应用，采用 **AI 驱动** 的数据分析引擎，帮助用户通过自然语言对话快速获取数据洞察。产品中文名"万策"寓意"于万千数据中，定最优之策"。

---

## ✨ 核心功能

### 1. 智能 AI 数据分析助手
- **自然语言查询**：通过对话方式提问，无需编写 SQL 或代码
- **意图智能理解**：自动识别用户分析意图，提供精准的分析建议
- **多轮对话分析**：支持连续追问，深入挖掘数据价值
- **上下文感知**：理解对话历史，保持分析连贯性

### 2. 多源数据接入
支持丰富的数据源类型：

| 数据源类型 | 具体支持 |
|-----------|---------|
| **文件数据** | Excel (xlsx/xls)、CSV、JSON |
| **关系型数据库** | MySQL、PostgreSQL、SQLite |
| **NoSQL 数据库** | MongoDB、Redis |
| **云数据仓库** | Snowflake、Google BigQuery |
| **电商平台** | Shopify、BigCommerce、eBay、Etsy |
| **项目管理** | Jira (Cloud/Server) |
| **API 接口** | UAPI、自定义 REST API |

### 3. 智能数据洞察
- **自动数据剖析**：AI 自动分析数据结构、类型和质量
- **智能洞察推荐**：基于数据特征推荐分析维度
- **异常检测**：自动识别数据中的异常值和趋势
- **关联分析**：发现数据字段间的隐藏关联

### 4. 丰富的可视化能力
- **多种图表类型**：折线图、柱状图、饼图、散点图、热力图等
- **拖拽式仪表盘**：自由布局，自定义数据看板
- **实时数据刷新**：支持定时更新数据展示
- **交互式图表**：支持缩放、筛选、下钻分析

### 5. 专业报告导出
支持多种格式的专业报告导出：

| 格式 | 特点 |
|-----|------|
| **PDF** | 精美排版，支持图表、表格、智能摘要 |
| **Word** | 完整报告文档，可编辑修改 |
| **Excel** | 多工作表数据导出，含样式和公式 |
| **PowerPoint** | 演示文稿格式，适合汇报展示 |

### 6. 分析技能市场（Skills）
内置专业分析技能，即装即用：
- **群组分析 (Cohort Analysis)**：用户留存率分析、生命周期价值评估
- **销售漏斗 (Sales Funnel)**：转化流程追踪、流失点识别
- **支持 Anthropic Skills**：兼容 Claude 技能协议
- **MCP 协议支持**：支持 LLM 模型上下文协议

### 7. 企业级特性
- **多语言支持**：简体中文、English 完整国际化
- **本地离线分析**：数据导入本地 SQLite，保护数据隐私
- **自动缓存管理**：智能数据缓存，提升分析性能
- **使用统计分析**：分析用量追踪（需授权服务器）

## 🚀 技术特点

### 先进技术栈
- **前端**：React + TypeScript + Tailwind CSS + Wails 2.0
- **后端**：Go 语言高性能服务
- **AI 引擎**：支持多种 LLM 模型（Claude、OpenAI、Gemini、DeepSeek、GLM、Qwen 等）
- **桌面端**：跨平台支持（Windows、macOS、Linux）

### 智能架构设计
- **Eino 智能体框架**：基于 CloudWeGo Eino 的 Agent 架构
- **意图路由系统**：智能分类用户请求，选择最佳处理路径
- **记忆系统**：长短期记忆结合，持续学习用户偏好
- **分析路径追踪**：完整记录分析过程，支持分析回放

### 安全与隐私
- **本地数据处理**：核心分析在本地完成，数据不离境
- **凭证加密存储**：敏感信息本地加密保存
- **权限最小化**：数据库连接使用只读权限
