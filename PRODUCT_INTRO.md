# Vantagics (万策) - 产品功能与特点说明

## 产品概述

**Vantagics (万策)** 是一款新一代智能商业智能桌面应用，让每一位用户都能轻松驾驭数据，洞见商业全貌。无论您是数据分析新手还是资深从业者，万策都能帮助您快速从数据中发现价值、做出决策。

---

## 核心功能

### 1. 智能数据连接

- **多格式支持**：一键导入 CSV、Excel、JSON、SQLite 等常见数据格式
- **智能解析**：自动识别数据结构和类型，减少预处理工作
- **本地存储**：所有数据存储在本地，保护您的商业隐私

### 2. 自然语言分析

- **AI 驱动对话**：用日常语言提问，AI 自动生成分析结果
- **上下文理解**：AI 记住对话历史，支持复杂问题的延续分析
- **多模型兼容**：支持 OpenAI、Anthropic Claude、Ollama、DeepSeek 等主流大语言模型

### 3. 交互式仪表盘

- **拖拽式设计**：直观地创建和自定义数据看板
- **丰富图表**：集成 ECharts 图表库，支持多种可视化类型
- **实时更新**：数据变化即时反映在仪表盘上

### 4. 自动化洞察

- **智能发现**：自动识别数据趋势、模式和异常
- **业务建议**：基于分析结果提供可执行的业务建议
- **定期报告**：快速生成数据报告，支持 PDF/HTML 导出

---

## 产品特点

### 隐私优先，数据不出本地

万策采用桌面原生架构设计，您的敏感业务数据始终存储在本地设备，无需上传云端。对于需要更高隐私保护的用户，我们支持本地部署的 LLM 模型（如 Ollama），真正实现数据"零外泄"。

### 零门槛，零基础上手

区别于传统 BI 工具的复杂操作，万策秉承"即插即用"的设计理念：
- 无需编写 SQL 或 Python 代码
- 无需掌握复杂的数据分析技能
- 友好的向导式操作流程

### AI 原生架构

万策从底层架构就融入了 AI 能力：
- 自然语言直接生成可视化图表
- AI 自动推荐最佳分析视角
- 智能助手全程陪伴分析过程

### 跨平台桌面体验

基于现代跨平台框架开发，提供原生桌面应用体验：
- 响应迅速，无网络延迟
- 深色/浅色主题自动适配
- 更好的系统集成能力

## 技术架构

- **前端**：React 18 + TypeScript + Tailwind CSS
- **后端**：Go 1.23 + Wails v2 跨平台框架
- **AI 集成**：Eino 智能体框架
- **数据存储**：SQLite + 本地文件
- **图表引擎**：Apache ECharts
